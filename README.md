# 载熙AI 领航工具

![载熙AI 领航工具](https://admin7730.github.io/zaixiAI.gitHub.io/)

## 项目简介

载熙AI 领航工具是一款集成了AI智能对话、系统工具管理和网页浏览功能的综合工具箱应用。它将各种系统工具组织成分类清晰的工具箱，同时提供AI智能助手功能，帮助用户更高效地管理和使用系统工具。

## 主要功能

### 🤖 智能对话
- 支持多种AI模型，包括免费模型
- 自然语言交互，提供智能问答和辅助
- 聊天记录保存和加载功能

### 🧰 工具箱
- 处理器工具：CPU-Z、Core Temp、Prime95等
- 内存工具：Thaiphoon、MemTest、ZenTimings等
- 显卡工具：GPU-Z、GpuTest、NVIDIA Inspector等
- 硬盘工具：CrystalDiskInfo、AS SSD Benchmark、HD Tune等
- 显示器工具：UFO测试、在线屏幕测试、色域检测等
- 外设工具：鼠标测试、键盘测试、按键映射等
- 烤鸡工具：FurMark、CPU Burner等
- 游戏工具：游戏平台启动器、加速器等
- 其他工具：系统维护、实用工具等

### 🌐 网页浏览
- 内置网页浏览器，支持直接URL访问和搜索
- AI网页内容总结功能
- 网页下载功能

### 💻 CMD领航
- 智能生成CMD/PowerShell命令
- 一键运行生成的命令
- 命令复制和保存功能

## 技术特点

- **现代化界面**：采用深色主题，流畅的动画效果，响应式设计
- **多执行方式**：针对工具运行权限问题，实现多种执行方式
- **相对路径**：使用相对路径，确保应用可移植性
- **图标支持**：为每个工具类别和具体工具提供直观的图标
- **搜索功能**：支持工具搜索，快速找到所需工具
- **分类管理**：清晰的工具分类，便于管理和使用

## 安装方法

1. **下载安装程序**：从GitHub Releases页面下载最新版本的安装程序
2. **运行安装程序**：双击 `ZaiXiAI_Setup_1.0.0.exe` 开始安装
3. **选择安装选项**：
   - 安装目录（默认：`C:\Program Files\载熙AI 领航工具`）
   - 是否创建桌面图标
   - 是否开机自启动
4. **完成安装**：点击"完成"按钮，程序将自动启动

## 使用说明

### 切换功能模块
- 点击左侧导航栏的图标切换不同功能模块
- 💬：智能对话
- 💻：CMD领航
- 🌐：网页浏览
- 🧰：工具箱
- ⚙️：设置
- ℹ️：关于

### 使用工具箱
1. 点击左侧导航栏的"🧰"图标进入工具箱
2. 在顶部搜索框输入工具名称进行搜索
3. 点击分类按钮筛选工具类别
4. 双击工具图标运行工具

### 使用AI助手
1. 点击左侧导航栏的"💬"图标进入智能对话
2. 在底部输入框中输入您的问题
3. 选择合适的AI模型
4. 点击发送按钮或按Enter键发送消息

### 使用网页浏览器
1. 点击左侧导航栏的"🌐"图标进入网页浏览
2. 在顶部地址栏输入网址或搜索关键词
3. 点击"前往"按钮或按Enter键访问
4. 使用"后退"、"前进"、"刷新"按钮导航
5. 点击"总结"按钮获取网页内容的AI总结
